Signierung von Paketen konfigurieren

Um Installationspakete zu signieren, benötigen Sie ein  Developer ID Installer  Zertifikat, welches sich im Anmeldeschlüsselbund Ihres Macs befinden muss. 

 

Wie Sie  Developer ID  Zertifikate erhalten und installieren können, entnehmen Sie bitte der entsprechenden Dokumentation von Apple:  Create Developer ID certificates

Wenn Sie auf Ihrem Mac Xcode installiert haben, können Sie Xcode benutzen, um Zertifikate zu erstellen und zu verwalten.  Apple hat hierzu folgende Dokumentation bereitgestellt:  Manage signing certificates

 

Sobald Sie mindestens ein gültiges  Developer ID Installer  Zertifikat im Anmeldeschlüsselbund Ihres Macs installiert haben, wird Script2Pkg dieses im Menü  Entwicklerteam:  im Hauptfenster der Anwendung anzeigen.

Standardeinstellung wiederherstellen

Um die Standardeinstellung wiederherzustellen, entfernen Sie das  Developer ID Installer  Zertifikat aus dem Anmeldeschlüsselbund Ihres Macs.
