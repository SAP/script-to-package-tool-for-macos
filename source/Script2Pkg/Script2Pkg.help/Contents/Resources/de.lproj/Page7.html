Script2Pkg

Fehlerbehebung

Signierung kann nicht aktiviert werden.

Sollte die Option  Pakete signieren  deaktiviert sein, überprüfen Sie bitte, ob sich mindestens ein gültiges Developer ID Installer Zertifikat in Ihrem Anmeldeschlüsselbund befindet.

 

Beglaubigung kann nicht aktiviert werden.

Die Option  Pakete beglaubigen  kann aktiviert werden, sobald die Option  Pakete signieren  aktiviert wurde und eine unterstützte Version von Xcode oder der Kommandozeilentools für Xcode auf dem Mac installiert ist. Sollten Sie verschiedene Versionen der Kommandozeilentools für Xcode installiert haben, stellen Sie bitte sicher, dass Sie eine unterstützte Version mittels  xcode-select  im Terminal ausgewählt haben.

 

Wenn eine große Anzahl Pakete auf einmal erstellt wird, wird nicht für jedes Paket eine Mitteilung angezeigt.

Öffnen Sie bitte die Systemeinstellungen und wählen Sie den Bereich  Mitteilungen  aus. Wählen Sie Script2Pkg aus der Liste der Anwendungen aus und ändern Sie bitte die Art der Mitteilungen auf  Hinweise .

 

Die Beglaubigung von verzeichnisbasierten Paketen schlägt fehl, wenn das Verzeichnis andere Pakete oder Binärdateien enthält.

Wenn Sie ein Paket beglaubigen lassen wollen, welches andere Paket oder Binärdateien (Programme) enthält, stellen Sie bitte sicher, dass diese Pakete und Binärdateien signiert sind. Ist dies nicht der Fall, wird Apple die Beglaubigung Ihres Paketes ablehnen.
