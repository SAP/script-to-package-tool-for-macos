Verteilungspakete

Normalerweise handelt es sich bei den von Script2Pkg erstellten Installationspaketen um Komponentenpakete, aber Sie können Script2Pkg so konfigurieren, dass es Verteilungspakete anstelle von Komponentenpaketen erstellt. Verteilungspakete sind ein spezielles Format von Installationspaketen und Apple verlangt unter bestimmten Umständen ihre Verwendung anstelle von Komponentenpaketen. Weitere Informationen zu Verteilungspaketen finden Sie im  Glossar .

Wenn Sie Verteilungspaketen benötigen, gehen Sie bitte wie folgt vor:

 

1. Wählen Sie im Menü  Script2Pkg  den Menüpunkt  Einstellungen …  aus.

 

 

2. Wählen Sie die Option  Paketierung  aus.

3. Aktivieren Sie die Option  Verteilungspakete erstellen .

Standardeinstellung wiederherstellen

Um die Standardeinstellung wiederherzustellen, deaktivieren Sie bitte die Option  Verteilungspakete erstellen .
