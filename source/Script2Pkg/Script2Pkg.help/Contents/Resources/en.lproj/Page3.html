Script2Pkg

To create a payload-free package using this app, you will need two things:

 

  • The Script2Pkg app

  • A script

 

Once you have both of those items available, please use the procedure to create a payload-free package.

 

1.  Launch the app .

2. Click the  Select scripts  button.

 

3. In the window which appears, select the script(s) you want to use and click the  Build  button.

4. Installer package(s) will be created and stored in the same location as the script(s) which were selected. Each installer package will have the same name as the source script.
