Creating notarized packages

To create a notarized payload-free package using this app, you will need these things:

 

• The app

• The app configured for signing

• The app configured for notarization

• A script

 

Once you have all items available, please use the procedure to create a signed and notarized payload-free package:

 

1. Launch the app.

2. Select the  Sign packages  option.

3. Select the  Notarize packages  option.

4. Click the  Select scripts  button.

5. In the window which appears, select the script(s) you want to use and click the  Build  button.

6. Installer package(s) will be created and stored in the same location as the script(s) which were selected. Each installer package will have the same name as the source script.
